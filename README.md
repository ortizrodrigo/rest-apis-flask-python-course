# REST APIs with Flask and Python in 2024

This repository contains my progress through the **"REST APIs with Flask and Python in 2024"** course by [Jose Salvatierra Fuentes](https://jslvtr.com/).

## Course Overview

This course teaches how to build modern, secure, production-ready REST APIs using Python and Flask. It covers core REST principles along with authentication, database management, background tasks, and deployment. It also provides hands-on experience with JWT security, SQLAlchemy and Alembic, background processing, and deploying APIs using Docker and Render.com.

## Structure

Each folder corresponds to a section/module in the course.
